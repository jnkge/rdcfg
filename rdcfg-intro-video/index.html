前端 skills 与 codegraph 统一安装

rdcfg 把重复的宿主配置、skills 拷贝和代码图谱初始化，收进一个交互式 CLI 流程。

5支持宿主
3内置前端 skills
1命令启动
open project terminal
Step 01

不用记一串安装步骤

进入项目目录后，直接用 npx 拉起向导；全局安装后也可以直接运行 rdcfg。

npx rdcfg
command
入口很短一条命令进入完整安装向导。
Step 02

先决定安装作用域

全局安装适合个人环境；项目级安装可以跟着仓库走，方便团队共享同一套 skills。

全局共享 当前项目 可重复执行
scope
Step 03

选择语言与技术方向

向导会按语言/栈过滤可用能力。前端场景里，Vue、设计、组件库规范能一起装好。

Frontend
Python / Go / PHP / Flutter
后续可继续扩展 skill 包
language
Step 04

勾选需要的前端 skills

比如 Element Plus、frontend-design、vue-best-practices，按项目需要组合安装。

Element Plus Frontend Design Vue Best Practices
skills
按需组合每个 skill 都保留自己的 SKILL.md 与引用资料。
Step 05

一次选择多个 AI 宿主

ZCode、Claude Code、Codex CLI、Cursor、Trae 可以同时勾选，减少来回切配置的时间。

ZZCode
CCodex / Claude
TTrae / Cursor
hosts
Step 06

顺手连接 codegraph

rdcfg 可以继续安装/连接 codegraph，让代码图谱 MCP 和当前项目初始化跟 skills 一起完成。

install / connect codegraph
一个流程skills 与代码图谱不再分散安装。
Result

skills 已落到目标目录

安装完成后，可以在项目或宿主目录看到对应 skill 文件夹，团队协作时也更容易对齐能力清单。

element-plus-vue3 frontend-design vue-best-practices

Ready
